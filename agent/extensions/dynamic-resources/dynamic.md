# Dynamic Resources

Reload or list dynamic resources registered through the extension API.

## Commands

- `/dynamic list` — List all dynamically registered resources
- `/dynamic reload` — Re-trigger resource discovery

## Notes

- Resources are discovered at startup automatically
- Use `/dynamic reload` after modifying dynamic resource files
