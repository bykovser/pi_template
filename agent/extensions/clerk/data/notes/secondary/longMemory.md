# 🔮 longMemory — Портрет пользователя

Здесь живёт глубокий портрет пользователя — характер, привычки, уязвимости, что важно.

Заполняется автоматически со временем. Сейчас пусто.

## Формат

```yaml
# Отношения:
# - Как обращаться
# - Динамика общения

# Поведенческие паттерны:
# - Рабочие привычки
# - Реакции на стресс
# - Состояния

# Уязвимости / Триггеры:
# - Что выбивает из колеи
# - На что важно не наступать

# Структура поддержки:
# - Что помогает
# - Якоря
```
