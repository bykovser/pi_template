# 📁 Clerk Template — Скелет для твоего агента

Этот скелет скопирован с живого Clerk (Жанночка), но **вычищен от личных данных**.

## 🤖 Что такое Clerk?

**Clerk** — это AI-ассистент с характером, который живёт внутри терминала и умеет:

- **Кодить, шалить на вашем компе с файлами** - построен на базе CLI агента [pi](https://github.com/nicepkg/pi)
- **Помнить** — RAM буфер (контекст разговора), midMemory (текущий контекст), longMemory (портрет пользователя, арки, факты), diary (личный дневник клерка), secondary notes (глубокие экспорты)
- **Спать** — sleep consolidation: переосмысление, компрессия памяти, поиск противоречий (как человеческий мозг)
- **Писать в Telegram** — через Bot API, с HTML-форматированием и файлами
- **Слушать гс из Telegram** — обычный шиндовс wisper
- **Смотреть картинки из Telegram** — через модель в openrouter
- **Смотреть картинки из терминала** — через модель в openrouter
- **Искать в интернете** — через DuckDuckGo (без API ключа)
- **Читать веб-страницы** — через Mozilla Readability
- **Дёргать Claude Code** — как субагента для сложных задач
- **Управлять задачами** — todo-лист, напоминалки, расписание
- **Иметь характер** — profile.yaml определяет личность, тон, правила (обязательно отредачить под себя (наверняка у вас уже есть какой-нибудь __домашний__ чат в какой-нибудь нейронке))
- **Быть проактивной** — сама предлагает помощь, задаёт вопросы, ставит напоминалки
- **Недавно мы с Жанночкой сломали само-перезагрузку)** - почините сами, тогда она сможет дописывать себя вообще не трогая вас

**Технически:** Clerk — это расширение для [pi](https://github.com/nicepkg/pi) (coding agent). Написана на TypeScript, работает как инжект в system prompt pi. Использует ту же LLM, что и pi.

## 🚀 Быстрый старт

### 1. Впиши API ключи
Открой `agent/auth.json` и вставь свои ключи:
```json
{
  "openrouter": {
    "type": "api_key",
    "key": "sk-or-v1-ТВОЙ_КЛЮЧ"
  }
}
```

### 2. Настрой персонажа
Открой `agent/extensions/clerk/data/profile.yaml`:
- **personality** — имя, характер, тон (оставь пример или перепиши)
- **rules** — правила поведения (оставлены как пример)
- **telegram** — 👇 см. пункт 4
- **facts** — факты о тебе (можно оставить пустым `[]`)

### 3. Активируй Telegram (опционально)
В `profile.yaml` в разделе `telegram` укажи:
```yaml
telegram:
  botToken: "YOUR_BOT_TOKEN"    # от @BotFather
  chatId: 123456789             # твой chat_id
```

### 4. Запусти
```bash
pi
```
